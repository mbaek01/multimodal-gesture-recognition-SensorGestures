# SensorGestures Project Page

Static project page for **Interpretable Multimodal Gesture Recognition for Drone and Mobile Robot Teleoperation via Log-Likelihood Ratio Fusion**.

## Contents

- `index.html`: project page markup
- `assets/css/style.css`: page styling
- `assets/js/main.js`: navigation and BibTeX copy behavior
- `assets/images/`: teaser, hardware, method, efficiency, LLR, and poster images
- `assets/videos/`: gesture demonstration clips
- `assets/pdfs/`: project paper PDF

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

The repo is ready for GitHub Pages. Keep `.nojekyll` in the repository root so GitHub Pages serves files directly without Jekyll processing.
